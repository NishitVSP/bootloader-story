GRUB (Grand Unified Bootloader) is now loaded into memory. It's like a menu that lets you choose which operating system to boot!

GRUB's Job:

1. Reads its configuration file

2. Shows a menu (text or graphical)

3. Locates the kernel binary file

4. Loads kernel + initramfs into RAM

5. Passes control to the kernel

Modern Version: Most Linux systems now use GRUB2, which is more powerful and flexible!
